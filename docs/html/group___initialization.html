fpl_api bool fpl::InitPlatform ( const InitFlags  initFlags = InitFlags::All,
const Settings &  initSettings = DefaultSettings() 
)

Initializes the platform layer.

Parameters
initFlagsOptional init flags used for enable certain features, like video/audio etc. (Default: InitFlags::All)
initSettingsOptional initialization settings which can be passed to control the platform layer behavior or systems. (Default: Settings and configurations provided by DefaultSettings())
Note
ReleasePlatform() must be called when you are done! After ReleasePlatform() has been called you can call this function again if needed.
Returns
Returns true when the initialzation was successful, otherwise false. Will return false when the platform layers is already initialized successfully.
