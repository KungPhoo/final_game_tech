How to use FPL?

In one of your C++ translation units include this:

#define FPL_IMPLEMENTATION
#include "final_platform_layer.hpp"

You can then include this file in any other C++ source or header file as you would with any other header file.

Provide the typical main entry point with at least the initialization and release of the platform:

int main(int argc, char *args[]) {
// Initialize the platform
if (fpl::InitPlatform(fpl::InitFlags::All)) {
// your code goes here
// Release the platform
fpl::ReleasePlatform();
return 0;
} else {
return -1;
}
}

How to use FPL in multiple translation units?

To use FPL in multiple translation units, i recommend creating a separated "final_platform_layer.cpp" in the same directory FPL is located and define the implementation there only:

final_platform_layer.cpp:

#define FPL_IMPLEMENTATION
#include "final_platform_layer.hpp"


This way FPL implementation is compiled once and can be used everywhere.

What do i need to get started?

The only dependencies are built-in operating system libraries, a C++/11 compiler and the C-Runtime.
Depending on the compiler and platform - linking to one system library may be required:

Win32

  • Link against "kernel32.lib"

Linux

  • Link against "libld.so"
