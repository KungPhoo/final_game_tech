wchar_t *  wideDest,
const uint32_t  maxWideDestLen 
)

Converts the given 8-bit source ansi string with length in a 16-bit wide string.

Parameters
ansiSourceThe 8-bit source ansi string.
ansiSourceLenThe number of characters of the source wide string.
wideDestThe 16-bit destination wide string buffer.
maxWideDestLenThe total number of characters available in the destination buffer.
Note
Null terminator is included always. Does not allocate any memory.
Returns
Returns the pointer to the first character in the destination buffer or nullptr when either the dest buffer is too small or the source string is invalid.
