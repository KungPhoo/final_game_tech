Contribute

I am happy that you want to support this project!

I created this project alone, so i really appreciate any kind of contributions.

There are several ways to help this project:

  • Test the library for any hardware/platform/compiler configuration you have access to and report the result -> This helps a lot!
  • Report any bugs you find
  • Improving this documentation by sending in new tutorials/examples, point me to typos etc.
  • Write pull-requests for fixed bugs
  • Write new demos
  • Recommend this project
  • Give me feedback

And lastly if you want to contact me for another reason, just drop a message in some way (Github issue, Private message, Socials etc.).

Thanks a lot!
