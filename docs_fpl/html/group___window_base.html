Enables or disables fullscreen mode.

Parameters
valueSet this to true for changing the window to fullscreen or false for switching it back to window mode.
fullscreenWidthOptional fullscreen width in screen units. When set to zero the desktop default is being used. (Default: 0)
fullscreenHeightOptional fullscreen height in screen units. When set to zero the desktop default is being used. (Default: 0)
refreshRateOptional refresh rate in screen units. When set to zero the desktop default is being used. (Default: 0)
Returns
True when the window was changed to the desire fullscreen mode, false when otherwise.
